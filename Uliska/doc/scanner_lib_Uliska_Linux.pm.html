NAME

scanner/lib/Uliska/Linux.pm - Perl module for Uliska inventory scanner for Linux

TODO

SYNOPSIS

  use Uliska::Linux;
  my $os = Uliska->init("Linux");
  $os->run() if defined $os;

DESCRIPTION

Linux module implements Linux support for Uliska. It is top-level Uliska OS module.

Is is called and executed from uliska.pl script. Together with this module other modules are/can be called:

  - Uliska::Linux::<version_minor>
  - Uliska::Linux::<version_major>

Module executes commands from cfg/Linux.cfg

Module performs additional functions:

  - detects Linux distribution (i.e. RedHat, Debian etc)
  - loads and executes corrsponding module (Linux::Redhat, Linux::Debian)

Additional modules are responsible for loading distribution-specific libraries and executing lists of distribution-specific commands.
