Installation procedure

  • Clone from Github:

    git clone --recursive git@github.com:dmytro/nagira.git

  • Install bundle

    bundle install

  • Run test suite

    • First to test your package

      Run rspec tests in development and/or test environments. In this case Nagira will use proven to work files included with the distribution (in test/data directory).

      RACK_ENV=<development|test> rspec -fd

    • Test production

      RACK_ENV=production rspec -fd

  • Run application

    bundle exec ./app.rb

or

rackup -p <port>

  • Access

Go to URL: localhost:4567/status
