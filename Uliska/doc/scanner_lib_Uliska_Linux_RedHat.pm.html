NAME

scanner/lib/Uliska/Linux/RedHat.pm - Perl module for Uliska inventory scanner for RedHat (or CentOS) Linux

SYNOPSIS

  use Uliska::Linux::RedHat;
  Uliska->init("Linux/RedHat")->run();

DESCRIPTION

    - Called from Uliska::Linux;
    - Detects major and minor versions of RedHat OS;
    - Executes commands from lists: cfg/Linux/RedHat/<major> and
      cfg/Linux/RedHat/<major>/<minor>.

FUNCTIONS

detect_version

Returns RedHat distribution version as hash with keys: 'major' and 'minor'.
